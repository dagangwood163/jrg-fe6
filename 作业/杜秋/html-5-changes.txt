  Tag //标签	        Description     //说明

<header>	 //定义标题的文件或部分       Defines a header for the document or a section
<main>	 //定义文件的主要内容             Defines the main content of a document
<mark>	   //定义标记或突出显示的文本      Defines marked or highlighted text

<summary>	 //定义一个可见的标题为<details>元素     Defines a visible heading for a <details> element
<time>	   //定义日期/时间                         Defines a date/time
<wbr>	   //定义一个可能的换行                     Defines a possible line-break

HTML5 - New Attribute Syntax        //HTML5的 - 新的属性语法
HTML5 allows four different syntaxes for attributes.     //HTML5允许四种不同的语法的属性。

This example demonstrates the different syntaxes used in an <input> tag:     //这个例子演示了一个<input >标签中使用的不同语法：


 Type	          Example

Empty	        <input type="text" value="John" disabled>
Unquoted       	<input type="text" value=John>
Double-quoted	<input type="text" value="John Doe">
Single-quoted	<input type="text" value='John Doe'>

In HTML5, all four syntaxes may be used, depending on what is needed for the attribute.    //HTML5中，所有四个语法可以使用，这取决于所需要的属性。
