HTML5 New Elements 

New Elements in HTML5

Below is a list of the new HTML5 elements,and a description of what they are used for.

NEW Semantic/Structural Elements
新的 语义/结构 元素

HTML5 offers new elements for better documents structure:
HTML5提供了新的元素去满足更加清晰的文档结构

Tag>>
<article> Defines an article in the document.
定义文档中的一篇文章

<aside> Defines content aside from the page content
定义页面主内容旁边的副内容

<bdi> Defines a part of text that might be formatted in a different direction from other
定义一段文字可能和别的文字格式的方向不同

<details> Defines additional details that the user can view or hide
定义用户可以查看或者隐藏的额外内容

<dialog> Defines a dialog box or window
定义一个弹框或者一个窗口

<figcaption> Defines a caption for a <figure> element
为figure定义一个标题

<figure> Defines self-contained content,like illustrations,diagrams,photos
定义独立的内容，比如插图，图标，照片

<footer> Defines a footer for the document or a section
为文档或者章节定义一个结尾

<header> Defines a hearder for the document or a section
为文档或者章节定义一个开头

<main> Defines the main content of a document
定义文档的主要内容

<mark> Defines marked or highlighted text
定义标记或者高亮文本

<menuitem> Defines a command/menu item that the user can invoke from a popup menu
定义一个用户可以调用弹出的菜单的命令或者菜单

<meter> Defines a scalar measurement within a known range (a gauge)
定义一个度量衡的范围

<nav> Defines navigation links in the document
在文档中定义导航连接

<progress> Defines the progress of a task
定义一个任务的进程

<rp> Defines what to show in browsers that do not support ruby annotations
ruby 注释中使用，以定义不支持 ruby 元素的浏览器所显示的内容。

<rt> Defines an explanation/pronunciation of characters(for East Asian typography)
定义一个字符的解释/发音

<ruby> Defines a ruby annotation (for East Asian typography)
定义一个ruby注释

<section> Defines a section in the document
定义文档中的章节

<summary> Defines a visible heading for a <details> element
为details元素定义一个标题

<time> Defines a date/time
定义日期/时间

<wbr> Defines a possible line-break
定义一个可行的行中断

New Form Elements

Tag

<detalist> Defines pre-defined options for input controls
定义输入控件的预定义选项

<keygen> Defines a key-pair generator field (for forms)
定义一个密钥对生成字段(用于表单)

<output> Defines the result of a calculation
定义计算结果
